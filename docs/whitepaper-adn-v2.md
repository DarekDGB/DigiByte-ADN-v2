# ADN v2 – Whitepaper  
### Autonomous Defense Node for the DigiByte Quantum Shield

## 1. Mission
ADN v2 is DigiByte’s autonomous security enforcer.  
Where Sentinel AI v2 **detects**, ADN v2 **acts**.

It is the third layer of the full defense system:
DQSN → Sentinel AI v2 → ADN v2 → Wallet Guardian.

## 2. Threat Era 2025–2035
- rented hashrate attacks  
- cross‑chain deep reorg operations  
- automated mempool spam  
- quantum preimage threats  
- AI‑driven botnets  

## 3. ADN v2 Philosophy
- No forks  
- No consensus changes  
- No user key interaction  
- Pure defense and containment  
- Works alongside existing DigiByte nodes  

## 4. How ADN v2 Responds
When Sentinel AI v2 flags risk:

### ELEVATED
- peer soft‑filtering  
- validate reorg depth  

### HIGH
- PQC enforcement  
- suspicious tx throttling  
- region anomaly analysis  

### CRITICAL
- hardened mode  
- freeze unsafe tx propagation  
- isolate malicious peers  
- notify Wallet Guardian  

## 5. PQC Integration
ADN v2 can escalate:
- legacy-key freeze  
- mandatory PQC signing  
- early warning for Shor attacks  

## 6. The 4‑Layer Shield
1. DQSN — raw entropy + anomaly detection  
2. Sentinel v2 — AI‑driven threat classification  
3. ADN v2 — defense execution  
4. Wallet Guardian — user protection  

## 7. Why ADN v2 Matters
It turns DigiByte into the first blockchain with a **self‑healing defense loop**:
detect → classify → act → protect → adapt.
